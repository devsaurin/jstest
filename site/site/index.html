The cross-platform JavaScript test framework

jstest is a testing framework for JavaScript that’s designed to run on any platform with minimal effort. It provides a familiar RSpec-style API for organising tests that can run on the following platforms:

  • Web browsers: Chrome, Firefox, Internet Explorer, Opera, Safari
  • PhantomJS (both in the browser and the scripting runtime)
  • Server-side platforms: Node.js, Narwhal, RingoJS
  • Database shells: MongoDB
  • GUI environments: XULRunner, Adobe AIR
  • Shells: V8 shell, Rhino, SpiderMonkey, Windows Script Host

The framework can produce a variety of output formats and includes an API for adding new ones. It comes with plugins for several popular browser test runners, including:

  • Buster.JS
  • Karma
  • Teabag
  • Testem
  • Testling CI
  • TestSwarm
